H5提现

Git https://gitlab.maskpark.net/vue/website-withdraw

通过H5页面完成提现功能，入口为iOS 客服聊天提现页面，客户端会申请一个临时code，通过url传参方式打开h5页面， 页面拿到code做校验信息，返回用户账户信息做后续提现操作。

功能介绍：

  • 打开页面校验 code，获取用户信息
  • 功能包括提现红包，提现面具币，修改支付宝账号
  • 操作中如果出现code 校验失败情况，通过scheme 打开App重新申请一个 code 进行操作

H5 充值

Git https://gitlab.maskpark.net/vue/website-recharge

面具币网页支付，面具用户打开页面登录，选择使用支付宝充值面具币，支付成功完成充值操作

功能介绍：

  • 面具账号登录，支付宝支付
  • 资源存放阿里云，index.html登录，pay.html支付，pay-done.html支付成功
  • 根据浏览器userAgent 支付宝支付 pc mobile 有做区分
  • 用户登录信息保持到 sessionStorage 里面，关闭页面后会清除用户信息
